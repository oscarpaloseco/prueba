# ToggleGroup

shadcn/ui ToggleGroup. A row of toggle items sharing the Toggle styling — joined (spacing 0, shared border, end-only radius) or gapped. type single / multiple. The pressed set bakes from a comma index list; the preview is a genuine Radix ToggleGroup. Fully bound to shadcn semantic tokens.

## Usage example

```tsx
<ToggleGroup />
```
